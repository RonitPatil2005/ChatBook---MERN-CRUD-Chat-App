# ChatBook---MERN-CRUD-Chat-App
ChatBook is a **basic MERN stack CRUD application** built for learning purposes.  
It is **not a real-time chat app**. This project focuses on understanding **Create, Read, Update, and Delete (CRUD)** operations using MongoDB, Express, Node.js, and EJS.
